项目介绍
=============

这是一个用css3和jquery做的小动画，灵感来源于<a href="http://lizclimo.tumblr.com/">Liz</a>的漫画，希望大家喜欢！        
            
你可以用鼠标左右键控制刺猬的转动，嘻嘻！    

具体的设计过程请移步到我的博客：<a href="http://fangge-sun.blog.163.com">Mr.F</a>            
                
此项目仅运行于支持CSS3的浏览器中，如有不便，敬请见谅!                   
推荐浏览器：chrome\firefox\Safri\IE10+
